# NetCore (v0.3)
NetCore - is a message-based, all-in-one highly optimized networking utility, allowing you to use multiple transports within one session.

Right now is **not** ready for usage in production - please wait for a month or so.

## Design goals
Designed to be highly scalable and flexible in use.

Unifies message handling with upmost optimization.

Supports:
- UDP.
- TCP.
- UDP+TCP.
- Unix.
- SteamNetworking.
